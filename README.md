# How to run Gulp

## 1. Use correct node version (18.16.0)

### Recommend using node version manager

```
nvm use
```

## 2. Install gulp cli

```
npm install --global gulp-cli
```

## 3. Install Dependencies

```
npm install
```

## 4. Run gulp

```
gulp
```
